# **App Name**: Fleet Admiral

## Core Features:

- User Authentication: Secure user login and registration for different roles (Admin, Driver, Passenger, Shipper).
- Real-time Fleet Tracking: Display vehicles on a live map with real-time location updates and status indicators.
- Automated Dispatch: AI powered tool to suggest vehicle and driver assignment for pending jobs, using criteria such as proximity, availability, vehicle type and driver fatigue levels.
- Job Management: Create, assign, update, and track transportation jobs (passenger, freight, equipment).
- Fatigue Monitoring: Track driver duty hours and fatigue levels, providing alerts and recommending breaks.
- Reporting and Analytics: Generate reports on fleet utilization, driver performance, and operational efficiency.
- User Roles: Implement different user interfaces for driver (driver-web), administrator (admin-web), passenger (passenger-web), shipper (shipper-web) and backend applications (backend).

## Style Guidelines:

- Primary color: Deep indigo (#4F46E5) for a sense of trust and authority.
- Background color: Light gray (#F9FAFB) for a clean and professional look.
- Accent color: Teal (#14B8A6) to highlight key actions and elements.
- Body and headline font: 'Inter' (sans-serif) for a modern, machined, objective, neutral look.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use clear and consistent icons throughout the interface. Icons should be simple and easily recognizable.
- Use a grid-based layout to ensure consistency and responsiveness across different screen sizes.
- Use subtle animations and transitions to enhance the user experience.
